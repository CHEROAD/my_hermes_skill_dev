## 搜索关键词

### 匹配类型

| 类型 | 变量 | 关键词 |
|------|------|--------|
| 酒店 | `KEYWORDS["hotel"]` | 酒店、住宿、如家、汉庭、7天... |
| 高铁 | `KEYWORDS["train"]` | 高铁、火车、动车、12306、铁路... |
| 航空 | `KEYWORDS["airline"]` | 航空、机票、航班、国航、东航... |
| 打车 | `KEYWORDS["taxi"]` | 高德、滴滴、打车、出租... |

### 排除词

```python
EXCLUDE_KEYWORDS = ["作废", "红字", "退货", "退款", "支付通知", "退票通知", "改签通知", "候补订单", "兑现成功"]
```

### 匹配优先级

匹配顺序：打车 → 酒店 → 航空 → 高铁。"电子发票"是通用词放高铁关键词，但打车/酒店含更具体词会先命中。

### 附件筛选

| 类型 | 保留 | 排除 |
|------|------|------|
| 酒店 | `dzfp_*.pdf` | 结账单 |
| 高铁 | ZIP 解压后 `.pdf` | `.ofd` |
| 高德打车 | 发票.pdf + 行程单.pdf | `.ofd` `.xml` |

### 匹配优先级

匹配顺序：打车(taxi) > 酒店(hotel) > 航空(airline) > 高铁(train)

原因：高铁关键词含"电子发票"等通用词，高德打车标题也会含"发票"，需先匹配打车避免误分类。

## 日期范围

默认扫描近 3 个月（90 天）：

```python
days = 90  # 可修改为其他天数
```

## 临时文件

- 默认使用 `tempfile.mkdtemp()` 创建临时目录
- 脚本结束时自动清理

## PDF 文件

- 默认保存到临时目录
- 文件名格式: `出差发票汇总_{日期}.pdf`

## 自定义修改

如果需要修改关键词或配置，编辑 `scripts/travel_invoice.py` 文件中的对应变量。
